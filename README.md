# Instagram Scraper

This repository provides a starting structure for developing a software project.

## Project layout

```
instagram-scraper/
├── src/
│   └── instagram_scraper/
│       ├── __init__.py
│       └── main.py
└── tests/
    └── test_main.py
```

Run the sample code:

```bash
python -m instagram_scraper.main
```

Run the tests with:

```bash
pytest
```
